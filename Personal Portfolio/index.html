Image to Text Conversion

The project aims to convert Text to picture pixels. For example if we gave text like “Flower with pink petals” by using GAN it generates an image that is most similar to the text. Text-to-image synthesis is all about converting text descriptions into appropriate images. Text-to-image synthesis refers to the method of generating images from the input text automatically. Deciphering data between pictures and text is a major issue in artificial intelligenc. Automatic image synthesis is highly beneficial in many ways. Generation of the image is one of the applications of conditional generative models. For generating images, GAN(Generative Adversarial Models) are used. Also, there is one problem with this project is that there are can be many probable outcomes for single text descriptions but it can be overcome by training the model.
